## 环境要求

Valhalla 需要满足以下环境要求

- Node.js >= 18
- PostgreSQL >= 13

## 安装 Node.js

Valhalla 需要 Node.js 作为运行环境, 请参考 [Node.js 官方网站](https://nodejs.org/) 下载并安装 Node.js。

## 安装 PostgreSQL

Valhalla 需要 PostgreSQL 作为数据库, 请参考 [PostgreSQL 官方网站](https://www.postgresql.org/) 下载并安装 PostgreSQL。
