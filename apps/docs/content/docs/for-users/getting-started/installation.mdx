import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>
 
### 准备工作

请阅读 [准备工作](/docs/for-users/getting-started/requirements) 章节, 确保您的环境满足要求。

</Step>
 
<Step>
 
### 下载源码

<Tabs items={['Git', 'Curl']}>
<Tab value="Git">
```bash
git clone https://github.com/sacredcraft/valhalla.git
```

</Tab>

<Tab value="Curl">
```bash
curl -LO https://github.com/sacredcraft/valhalla/archive/refs/heads/main.zip
```

</Tab>
</Tabs>

</Step>

<Step>

### 安装依赖

进入项目根目录, 执行以下命令安装依赖

```package-install
npm install
```

</Step>

<Step>

### 项目配置

请阅读 [项目配置](/docs/for-users/getting-started/config) 章节, 配置项目所需参数。

</Step>

<Step>

### 启动项目

在项目根目录执行以下命令启动项目

```package-install
npm run start
```

</Step>

</Steps>
