---
title: 基本概念
---

Valhalla 是一个基于 `资源` 的资源管理平台, 我们按 `资源` 的类型, 对资源进行分类管理,
随后再根据资源的不同文件, 让文件匹配到对应的模版, 最后根据模版管理对应的文件。

## 资源

任何一个属于计算机内的文件夹, 都可以被视为一个资源。

下面列举几个常见的资源:

- 桌面资源: `C:\Users\Administrator\Desktop`
- 文档资源: `C:\Users\Administrator\Documents`
- 下载资源: `C:\Users\Administrator\Downloads`
- 图片资源: `C:\Users\Administrator\Pictures`

展开想象力, 还可以有:

- 美术文件资源: `C:\Users\Administrator\Desktop\美术文件`
- 代码文件资源: `C:\Users\Administrator\Desktop\代码文件`

此外, 每一个资源都需要用户有相应的权限才能访问。

## 模版

模版将匹配不同的文件, 如图片模版能够匹配图片文件, 代码模版能够匹配代码文件。
而不同的模版胜任不同的能力, 如图片模版能够编辑图片, 代码模版能够编辑代码。

模版是 Valhalla 的核心, 是 Valhalla 能够管理资源的基础, 它具有十分强大的拓展性。

- 你可以使用可视化的图片编辑模版, 来编辑图片。
- 你可以使用如 `Monaco Editor` 的代码编辑模版, 来编辑代码。
- 你可以使用如 `Markdown Editor` 的 Markdown 编辑模版, 来编辑 Markdown 文件。
- 你还可以通过可视化的表单, 来编辑特定的文件, 常用于 `yaml`, `json`, `toml` 等文件的编辑。

我们为模版赋予了十分强大的协作能力, 你可以在模版中添加多人协作编辑的功能, 让多个用户可以在同一个模版中编辑同一个文件。

## 文件

文件是 Valhalla 中最重要的概念, 是 Valhalla 中所有操作的载体。
