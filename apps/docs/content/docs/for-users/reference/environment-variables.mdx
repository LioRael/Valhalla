## `DATABASE_URL`

数据库连接地址

```bash
DATABASE_URL=postgresql://postgres:password@localhost:5432/valhalla
```

## `AUTH_SECRET`

用户认证密钥, 它应当是一个至少 32 个字符的随机字符串。

你可以使用以下命令生成一个随机字符串 (我们建议你使用 `openssl`)

使用 `CLI` 工具会自动将密钥写入 `.env.local` 文件。
你需要将其添加到 `.env` 文件中并删除 `.env.local` 文件。

<Tabs items={["Bash", "CLI"]}>
<Tab value="Bash">
```bash
openssl rand -base64 33
```
</Tab>

<Tab value="CLI" className="p-0">

<Tabs items={["npm", "pnpm", "yarn", "bun"]} className="rounded-none border-none">

<Tab value="npm">
```bash
npx auth secret
```
</Tab>

<Tab value="pnpm">
```bash
pnpx dlx auth secret
```
</Tab>

<Tab value="yarn">
```bash
yarn dlx auth secret
```
</Tab>

<Tab value="bun">
```bash
bun auth secret
```
</Tab>

</Tabs>

</Tab>

</Tabs>

下面是一个示例:

```bash
AUTH_SECRET=X3K131LCQyonfh7lv5KaWCkNwQ00MiDPnNqhVRan1EI=
```

## `AUTH_TRUST_HOST`

如果需要允许外网访问, 请设置 `AUTH_TRUST_HOST` 为 `true`

```bash
AUTH_TRUST_HOST=true
```
